- **Framework:** React  
- **Build Tool:** Vite  
- **API Client:** Apollo Client  
- **Routing:** React Router  
- **Styling:** Tailwind CSS, SASS/SCSS  
- **Language:** TypeScript  
- **State Management:** React Context API

## 🆘 Troubleshooting Quick Help

### Site won't build?
```bash
rm -rf _site .jekyll-cache
bundle install
bundle exec jekyll serve
```

### Build fails after push?
1. Check GitHub Actions tab for errors
2. Verify all YAML syntax
3. Wait 5-10 minutes for retry

### Navigation missing?
- Verify `_data/navigation.yml` exists
- Check YAML indentation (no tabs)
- Refresh browser cache

### Logs not showing?
- Verify folder name: `_logs/` (with underscore)
- Check file format: `YYYY-MM-DD-day-NNN.md`
- Verify front matter YAML syntax
